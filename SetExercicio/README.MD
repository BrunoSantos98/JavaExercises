## 💻 Pré Requisitos 💻:

- Sistema operacional: Qualquer Distribuição Linux, Windows e Mac OS

compilar java.

## 📃 Como usar:

  2. abra na sua IDE ou editor de preferencia

  3. Teste como quiser.
